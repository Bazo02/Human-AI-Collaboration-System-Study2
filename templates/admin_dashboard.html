What each log contains
Participants: one row per participant with metadata, condition order, and completion status.
Decisions: one row per case decision (choice, correct/incorrect, time, AI seen/followed, AI recommendation/confidence/probability).
Events: interaction events (clicks, openings, page actions) with optional JSON payload.
Surveys: post-task questionnaire answers stored as JSON (one row per participant).
